Windows

What is Windows

Windows is an operating system developed by Microsoft. It is a software that manages all the operations of a computer and allows the user to interact with the hardware and with other programs.

Windows gets its name from its graphical interface, which is based on "windows" - separate display areas on the screen that contain different applications and files.

When and where it was created, designed, and launched

Windows was designed in 1981 in the United States of America, initially as a graphical interface for MS-DOS (the command-line operating system). It was improved in 1983 under the name Interface Manager and completed in its first version (Windows 1.0) in 1985.

Windows was officially launched on November 20, 1985, at Microsoft's headquarters in Bellevue, Washington (USA).

Who invented it

Windows was invented by a team of engineers and programmers at Microsoft, based on an idea from the company's two founders, Bill Gates and Paul Allen. Bill Gates is an American computer scientist who, starting from the dream that by the year 2000 every home would have a computer and that everyone should be able to use it, was the main promoter of the idea of creating a graphical interface to make PCs more accessible.

Paul Allen (who passed away in 2018) was an American computer scientist and the main promoter of the founding of Microsoft.

Main features

Graphical User Interface (GUI)

Windows is based on a graphical interface made up of icons, windows, menus, and buttons, each corresponding to underlying textual commands.

Window-based structure

Windows is designed around a window-based structure, which means that information and programs are displayed in separate windows that can be opened and managed simultaneously, making multitasking easier.

Universal software and hardware compatibility

Windows is the most widely used operating system in the world, so most programs and peripherals are designed to work with it.
